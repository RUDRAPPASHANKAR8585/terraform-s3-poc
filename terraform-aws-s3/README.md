# terraform-aws-s3

## Usage

```bash
terraform init
terraform fmt
terraform validate
terraform plan
terraform apply
terraform destroy
```

## Features

- Bucket Configuration
- Encryption
- Versioning
- Ownership Controls
- Public Access Block
- Server Access Logging
- Website Hosting
- Event Notifications
- Transfer Acceleration
- Requester Pays
- Object Lock
- Bucket Policies
